Birthday Card Generator

An application that allows a user to make a birthday card featuring a fun item related to the recipient's date of birth.

When the user enters the birthday of the intended recipient and chooses a card theme, they are presented with an automatically generated birthday card featuring appropriate styling and relevant imagery. I was a core collaborator in a group of four.

Snack Time

A simple web application that lets the user add and eat their favorite foods.

The user is able to add their own snacks. They are asked for the snack's name, to decide if the snack is healthy or unhealthy, and then submit it to an 'uneaten' list. When the user wishes to eat the snack, they can click 'devour it!' to move it to the devoured column. This lets them track what they've eaten.

Bored Jar

A mobile first application that helps users decide how to spend their free time.

This application lets users add quick tasks to a 'bored-jar'. When the user has free time on their hands, the application will present them with a random task based on the length of the activity. I was a key contributor in a group of four.
